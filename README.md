
1. ТРЕБОВАНИЯ
====================================================
- Python 3.12
- Telegram Bot Token (получить у @BotFather)
- Telegram ID (узнать у @userinfobot)


2. УСТАНОВКА И ЗАПУСК
====================================================

ШАГ 1: создать проект
----------------------------------------------------
создай папку и закинь туда файлы с ботом

ШАГ 2: Создай виртуальное окружение
----------------------------------------------------
(команды нужно прописывать в терминале
в vscode он окрывается на ctrl+j)

Windows:
python -m venv venv

venv\Scripts\activate

Linux/Mac:
python3 -m venv venv
source venv/bin/activate


<<<<<<< HEAD
ШАГ 3: Установи зависимости
----------------------------------------------------
pip install -r requirements.txt


ШАГ 4: Создай файл .env
----------------------------------------------------
Создай файл .env в папке с ботом и вставь:

BOT_TOKEN=сюда_вставь_токен_от_botfather
ADMIN_ID=сюда_вставь_свой_id

Если админов несколько:
ADMIN_ID=123456789,987654321,555666777


ШАГ 5: Запусти бота
----------------------------------------------------
python bot.py

Успешный запуск выглядит так:
✅ Загружено админов: 2
✅ ID админов: [123456789, 987654321]
✅ Бот инициализирован
✅ Курсы обновлены (#1)


3. ГДЕ МЕНЯТЬ НАСТРОЙКИ
====================================================

Открой файл bot.py и найди класс Config (Ctrl+F)

----------------------------------------------------
class Config:
    # Суммы в рублях
    MIN_SUM_RUB = 500        # Минимальная сумма сделки
    MAX_SUM_RUB = 1_000_000   # Максимальная сумма сделки
    
    # Наценка бота (%)
    BUY_MARKUP = 16    # Наша комиссия при продаже клиенту
    SELL_MARKUP = 15   # Наша комиссия при покупке у клиента
    
    # Комиссии сети (ОЧЕНЬ ВАЖНО!)
    NETWORK_FEES = {
        'USDT': 1.0,      # Комиссия за USDT
        'BTC': 0.00002,   # Комиссия за BTC
        'LTC': 0.02,      # Комиссия за LTC
        'TON': 1.0        # Комиссия за TON
    }
    
    # Защита от флуда
    RATE_LIMIT_MESSAGES = 100   # Сколько сообщений можно
    RATE_LIMIT_PERIOD = 60      # За сколько секунд
    BLOCK_TIME = 60             # Блокировка на сколько секунд
----------------------------------------------------


4. ГДЕ МЕНЯТЬ ТЕКСТЫ
====================================================

В том же файле bot.py найди (Ctrl+F):

WELCOME_TEXT        # Приветствие для пользователей
ADMIN_WELCOME_TEXT  # Приветствие для админа
HELP_TEXT          # Текст помощи
CONTACTS_TEXT      # Контакты

Просто меняй текст между кавычками на свой.



5. мини гайд
====================================================

1. Скачал Python
2. Открыл терминал в папке с ботом
3. Ввел по порядку:

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

4. Создал файл .env с токеном и ID
5. Ввел: python bot.py
